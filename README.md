# 🐂 QurbaniHat - Digital Livestock Marketplace

QurbaniHat is a modern, responsive livestock marketplace designed for the Qurbani season. It allows users to browse various animals, view detailed information, and manage their profile.

---

## 🚀 Live Demo

**[Click here to visit the live site](https://a8-liard.vercel.app)**

---

## 🎯 Purpose

The purpose of this project is to simplify the traditional process of buying Qurbani animals by providing a digital platform. It ensures a seamless experience with secure authentication and a user-friendly interface.

---

## ✨ Key Features

- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
- **Secure Authentication:** Powered by **Better Auth** (Email/Password & Google Login).
- **Private Routes:** Dashboard and profile access are restricted to logged-in users only.
- **Form Management:** Fast and validated forms using **React Hook Form**.
- **Real-time Notifications:** Feedback via **React Hot Toast**.
- **Smooth Animations:** Integrated with **Animate.css**.

---

## 🛠️ Tech Stack & Packages

- **Framework:** Next.js 15 (App Router)
- **Database:** MongoDB
- **Authentication:** Better Auth
- **Styling:** Tailwind CSS
- **Form Handling:** React Hook Form
- **Icons:** React Icons
- **Toast Notifications:** React Hot Toast
- **Animations:** Animate.css

---

## ⚙️ Local Setup (For Developers)

If you want to run this project on your machine:

1. Clone the repo: `git clone https://github.com`
2. Install dependencies: `npm install`
3. Add `.env.local` with your MongoDB and Auth credentials.
4. Run: `npm run dev`
